# L1 Answers

### Expand (Positive)
- $(x + 4)^2 = x^2 + 8x + 16$
- $(2x + 3)^2 = 4x^2 + 12x + 9$
- $(3x + 5)^2 = 9x^2 + 30x + 25$

### Expand (Negative)
- $(x - 6)^2 = x^2 - 12x + 36$
- $(2x - 7)^2 = 4x^2 - 28x + 49$
- $(4x - 3)^2 = 16x^2 - 24x + 9$

### Simplify (Positive)
- $x^2 + 10x + 25 = (x + 5)^2$
- $x^2 + 14x + 49 = (x + 7)^2$
- $4x^2 + 24x + 36 = (2x + 6)^2$

### Simplify (Negative)
- $x^2 - 8x + 16 = (x - 4)^2$
- $x^2 - 12x + 36 = (x - 6)^2$
- $9x^2 - 24x + 16 = (3x - 4)^2$
