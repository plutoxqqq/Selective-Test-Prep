# L3 Answers

### Factorise

- $x^2 + 4x + 3x + 12$
  
  $(x+3)(x+4)$
  
- $2x^2 + 5x + 4x + 10$
  
  $(x+2)(2x+5)$
  
- $3x^2 - 6x + 2x - 4$
  
  $(3x+2)(x-2)$
  
- $4x^2 + x - 16x - 4$
  
  $(x-4)(4x+1)$
  
- $6x^2 - x - 12x + 2$
  
  $(x-2)(6x-1)$
