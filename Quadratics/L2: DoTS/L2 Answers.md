# L2 Answers

### Expand
- (x+6)(x-6) = x^2 - 36
- (2x+9)(2x-9) = 4x^2 - 81
- 5(3x+4)(3x-4) = 5(9x^2 - 16) = 45x^2 - 80

### Simplify
- x^2 - 49 = (x - 7)(x + 7)
- 4x^2 - 25 = (2x - 5)(2x + 5)
- 6x^2 - 150 = 6(x^2 - 25) = 6(x - 5)(x + 5)
